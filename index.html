neurocl

Neural Network C++ implementation

Intro

  • Neurocl = Neuro Computing Library)
  • meant to experiment various 'from scratch' implementations of neural network layer schemes
  • focuses on matrix expression of feed forward/backward propagation steps
  • implements Multi-Layer Perceptron (MLP) scheme, as well as Convolutional Neural Network (CONVNET) scheme

MLP Implementations

There are two different MLP implementations in Neurocl :

  • one using standard Boost.ublas containers
  • one based on VexCL containers

CONVNET Implementations

There is only one CONVNET implementation for now, based on a tensor abstraction class, using Boost.ublas containers.
